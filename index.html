Up Notification

JQuery library for simple and easy web notifications

Donwload »

Requeriments

You must set JQUERY and FONT-AWESOME in your project.

Get started

  1. Copy the JS and CSS files into your directory project.
  2. In the <head> of your html, reference the location to your upnotification.css.

    3. <link rel="stylesheet" href="ruta/css/upnotification.css">

  3. In the very bottom of your <body> html tag, reference the location to your upnotification.js.

    4. <script href="ruta/js/upnotification.js"></script>

  4. Just create a new instance of UpNotification and call the create method.

<link rel="stylesheet" href="ruta/css/upnotification.css">

<script href="ruta/js/upnotification.js"></script>

var notificacion = new UpNotification({

});

notificacion.create();

Custom creation

Create a new instance of UpNotification with anonimous object for settings configurations. Customice the UpNotification as best for your project.

var notificacion = new UpNotification({

titulo: 'Lorem',

mensaje: 'Lorem ipsum dolor sit amet, consectetur adipisicing elit. Sequi necessitatibus libero ea quis? Earum minima qui nostrum ullam officiis facere!'

});

notificacion.create();

Options

posicion

You can set a new position by:

notificacion.setPosicion(4);

var notificacion = new UpNotification({

posicion:2

});

notificacion.create();

icono

Set de icon only with the name of the font-awesome icon class like 'fa-star'

ICONOS

You can set a new icon by:

notificacion.setIcono('fa-camera');

var notificacion = new UpNotification({

icono:'fa-star'

});

notificacion.create();

titulo

Text for title of the UpNotification

You can set a new title by:

notificacion.setTitulo('New title');

var notificacion = new UpNotification({

titulo:'Attention!'

});

notificacion.create();

mensaje

Text for the message of the UpNotification

You can set a new message by:

notificacion.setMensaje('New message');

var notificacion = new UpNotification({

mensaje: 'Lorem ipsum dolor sit amet consectetur adipisicing elit. Blanditiis placeat cumque perferendis! Ipsam facere corporis quasi ab unde incidunt reprehenderit?'

});

notificacion.create();

color

Set the icon and border color with hexadecimal notation.

You can change this color by:

notificacion.setColor('#FF00FF');

var notificacion = new UpNotification({

color: ''

});

notificacion.create();

background

Set the color for the background with hexadecimal notation.

You can change the background color by:

notificacion.setBackground('#FF00FF');

var notificacion = new UpNotification({

background: ''

});

notificacion.create();

colorTexto

Set the color text of the UpNotification

You can change the color text by:

notificacion.setColorTexto('#FF0000');

var notificacion = new UpNotification({

colorTexto: ''

});

notificacion.create();

time

Time in miliseconds for hide the UpNotification, by default the UpNotification doesn't hide

You can change this time by:

notificacion.setTime(5000);

var notificacion = new UpNotification({

time: 3000

});

notificacion.create();

callback

Callback to execute when the UpNotification hide

You can set a new callback by:

notificacion.setCallback(function(){alert("Fire!");});

var notificacion = new UpNotification({

callback: function(){

alert("Fire!");

}

});

notificacion.create();
