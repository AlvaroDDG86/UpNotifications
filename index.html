Up Notification

Libreria JQuery de notificaciones web

Donwload »

Requerimientos

Es necesario tener JQUERY cargado, además de FONT-AWESOME para los iconos de la notificación.

Fácil de usar

Descargar la librería y añadir las referencias:

CSS

<link rel="stylesheet" href="ruta/css/upnotification.css">

Javascript

<script href="ruta/js/upnotification.js"></script>

Crea una instancia de UpNotification y ejecuta su método create();

<link rel="stylesheet" href="ruta/css/upnotification.css">

<script href="ruta/js/upnotification.js"></script>

var notificacion = new UpNotification({

});

notificacion.create();

Personalización en la creación

Crea una instancia de UpNotification, y con un objeto anónimo, configura la notificación como más se adapte a tu proyecto

var notificacion = new UpNotification({

titulo: 'Lorem',

mensaje: 'Lorem ipsum dolor sit amet, consectetur adipisicing elit. Sequi necessitatibus libero ea quis? Earum minima qui nostrum ullam officiis facere!'

});

notificacion.create();

Posibles opciones

posicion

Puedes cambiar la posición mendiante la función:

notificacion.setPosicion(4);

var notificacion = new UpNotification({

posicion:2

});

notificacion.create();

icono

Solo necesitas especificar el nombre del icono de font-awesome que quieres mostrar en la notificación

ICONOS

Puedes cambiar el icono mendiante la función:

notificacion.setIcono('fa-camera');

var notificacion = new UpNotification({

icono:'fa-star'

});

notificacion.create();

titulo

Cadena de texto que se mostrará en el título de la notificación

Puedes cambiar el título mendiante la función:

notificacion.setTitulo('Nuevo título');

var notificacion = new UpNotification({

titulo:'¡Atención!'

});

notificacion.create();

mensaje

Cadena de texto que se mostrará en el mensaje de la notificación

Puedes cambiar el mensaje mendiante la función:

notificacion.setMensaje('Nuevo mensaje');

var notificacion = new UpNotification({

mensaje: 'Lorem ipsum dolor sit amet consectetur adipisicing elit. Blanditiis placeat cumque perferendis! Ipsam facere corporis quasi ab unde incidunt reprehenderit?'

});

notificacion.create();

color

Seleccionar el color del icono y del borde de la notificación

Puedes cambiar el color mendiante la función:

notificacion.setColor('#FF00FF');

var notificacion = new UpNotification({

color: ''

});

notificacion.create();

background

Seleccionar el color del background de la notificación

Puedes cambiar el color del background mendiante la función:

notificacion.setBackground('#FF00FF');

var notificacion = new UpNotification({

background: ''

});

notificacion.create();

colorTexto

Seleccionar el color del mensaje y el título de la notificación

Puedes cambiar el color del texto mendiante la función:

notificacion.setColorTexto('#FF0000');

var notificacion = new UpNotification({

colorTexto: ''

});

notificacion.create();

time

Tiempo en milisegundos desde que se crea la notificación hasta que desaparece, por defecto el tiempo está deshabilitado.

Puedes cambiar el tiempo mendiante la función:

notificacion.setTime(5000);

var notificacion = new UpNotification({

time: 3000

});

notificacion.create();

callback

Función que se ejecutará cuando desaparezca la notificación

Puedes cambiar la función mendiante:

notificacion.setCallback(function(){alert("Fire!");});

var notificacion = new UpNotification({

callback: function(){

alert("Fire!");

}

});

notificacion.create();
